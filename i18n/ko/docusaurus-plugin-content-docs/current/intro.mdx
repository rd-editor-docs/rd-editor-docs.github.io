# 소개
<h3 className="subtitle">리듬 닥터가 되고 싶으신가요? 그러려면 상당한 연습이 필요할 겁니다.</h3>

[Rhythm Doctor](https://store.steampowered.com/app/774181/Rhythm_Doctor/)는 Steam에 정식 출시되었으며, 게임에 **레벨 에디터**가 포함되어 있습니다.  
다만, **무료 레벨 에디터**는 여전히 [이곳](https://giacomopc.itch.io/rdle)에서 이용할 수 있습니다.

<img className="left" src={useBaseUrl('/img/download.png')} alt="itch.io 다운로드 페이지"/>

레벨 제작에 도움이 되는 유용한 자료들은 다음과 같습니다.

- [Audacity](https://www.audacityteam.org/download/)
- [BPM(분당 박자 수)를 계산해 주는 도구 (예: All8)](http://www.all8.com/tools/bpm.htm)
- [애니메이션 이징(easing) 목록](https://easings.net/)
- [youtube-dl: 다양한 사이트에서 오디오를 다운로드할 수 있는 도구](https://ytdl-org.github.io/youtube-dl/index.html)
- [Rhythm Doctor 디스코드 서버](https://discord.gg/rhythmdr)
- [커스텀 레벨 목록](https://rhythm.cafe/)

이 가이드북은 **Klyzx, WillFlame, huantian**이 작성했습니다.
