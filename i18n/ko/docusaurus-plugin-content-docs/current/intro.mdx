# 도입부
<h3 className="subtitle">리듬 닥터가 되고 싶으신가요? 엄청난 연습이 필요할 거예요.</h3>

[리듬 닥터](https://store.steampowered.com/app/774181/Rhythm_Doctor/)는 스팀에 정식 출시되었고, 레벨 에디터를 포함하고 있습니다. 다만, 무료 레벨 에디터를 여전히 [여기](https://giacomopc.itch.io/rdle)에서 다운로드할 수 있습니다.

<img className="left" src={useBaseUrl('/img/download.png')} alt="itch.io downloads page"/>

레벨 제작에 도움이 될만한 여러 사이트들:
- [Audacity](https://www.audacityteam.org/download/)
- [All8, BPM(분당 박자 수)을 측정할 수 있는 툴](http://www.all8.com/tools/bpm.htm)
- [가감속 애니메이션 리스트](https://easings.net/)
- [youtube-dl, 여러 소스에서 오디오를 다운로드할 수 있는 툴](https://ytdl-org.github.io/youtube-dl/index.html)
- [리듬 닥터 라운지 디스코드](https://discord.gg/rhythmdr)
- [커스텀 레벨 리스트](https://rhythm.cafe/)
