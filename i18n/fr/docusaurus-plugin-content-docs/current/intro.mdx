# Introduction
<h3 className="subtitle">Donc, voulez-vous devenir un Rhythm Doctor? Cela va demander énormément de pratique.</h3>

[Rhythm Doctor](https://store.steampowered.com/app/774181/Rhythm_Doctor/) est sorti sur Steam et inclut l'éditeur de niveaux. Cependant, l'éditeur de niveaux gratuit est toujours disponible [ici](https://giacomopc.itch.io/rdle).

<img className="left" src={useBaseUrl('/img/download.png')} alt="page de téléchargements itch.io"/>

Quelques autres liens utiles pour créer des niveaux sont:
- [Audacity](https://www.audacityteam.org/download/)
- [Tout ce qui peut calculer le BPM, comme celui-ci sur All8](http://www.all8.com/tools/bpm.htm)
- [Une liste des courbes d'accélérations](https://easings.net/)
- [youtube-dl, un outil qui peut télécharger de l'audio à partir de nombreuses sources](https://ytdl-org.github.io/youtube-dl/index.html)
- [Le Discord du Rhythm Doctor](https://discord.gg/rhythmdr)
- [Liste des niveaux communautaires](https://rhythm.cafe/)
