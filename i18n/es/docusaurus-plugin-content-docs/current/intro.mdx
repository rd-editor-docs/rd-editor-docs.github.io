# Introducción
<h3 className="subtitle">Entonces quieres ser un Rhythm Doctor? Te llevará bastante práctica.</h3>

[Rhythm Doctor](https://store.steampowered.com/app/774181/Rhythm_Doctor/) ha sido publicado en Steam, e incluye el editor de niveles. Pese a eso, el editor gratuito es aún disponible [aquí](https://giacomopc.itch.io/rdle).

<img className="left" src={useBaseUrl('/img/download.png')} alt="Página de descarga itch.io"/>

Otros vínculos útiles para hacer niveles incluyen:
- [Audacity](https://www.audacityteam.org/download/)
- [Cualquier cosa que pueda calcular BPM, como este en All8](http://www.all8.com/tools/bpm.htm)
- [Una lista de suavizados en animación](https://easings.net/)
- [youtube-dl, una herramienta que puede descargar audio de muchas fuentes](https://ytdl-org.github.io/youtube-dl/index.html)
- [El Discord de Rhythm Doctor](https://discord.gg/rhythmdr)
- [Lista de niveles comunitarios](https://rhythm.cafe/)
