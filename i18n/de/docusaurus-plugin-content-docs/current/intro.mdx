# Einleitung
<h3 className="subtitle">Also, du möchtest ein Rhythm Doctor werden? Dazu ist eine Menge Übung erforderlich.</h3>

[Rhythm Doctor](https://store.steampowered.com/app/774181/Rhythm_Doctor/) wurde auf Steam veröffentlicht und enthält den Level-Editor. Der kostenlose Level-Editor ist [hier](https://giacomopc.itch.io/rdle) jedoch weiterhin verfügbar.

<img className="left" src={useBaseUrl('/img/download.png')} alt="itch.io Downloads Seite"/>

Einige andere nützliche Links zum Erstellen von Levels sind:
- [Audacity](https://www.audacityteam.org/download/)
- [Alles, was BPM berechnen kann, wie dieses hier auf All8](http://www.all8.com/tools/bpm.htm)
- [Eine Liste von Übergangsfunktionen](https://easings.net/)
- [youtube-dl, ein Tool, das Audio aus vielen Quellen herunterladen kann](https://ytdl-org.github.io/youtube-dl/index.html)
- [Das Rhythm Doctor Discord](https://discord.gg/rhythmdr)
- [Liste der Community-Level](https://rhythm.cafe/)
